## How to edit the website

You don't need any software. Everything is done directly on the live website.

### Step 1 — Open the website
Go to **https://dorecanineservices.co.uk**

### Step 2 — Open the editor
Press **Ctrl + E** on your keyboard (hold Ctrl, tap E)

A password box will appear.

### Step 3 — Enter the password
```
dorecanine2025
```

### Step 4 — Edit your content
Every piece of text on the page now has a dashed brown outline around it.

**Just click on any text and start typing** — it works exactly like editing a Word document.

You can edit:
- Prices
- Service descriptions
- Testimonials / reviews
- Contact details
- Any heading or paragraph

### Step 5 — Download your updated file
When you're happy with your changes, click the **"⬇️ Download updated site"** button in the brown bar at the top of the page.

This saves a file called **index.html** to your Downloads folder.

### Step 6 — Upload to GitHub to go live
1. Go to **https://github.com/alangner69/dcs**
2. Click on **index.html** in the file list
3. Click the **pencil icon ✏️** (top right of the file)
4. Click the **three dots ···** menu next to the pencil
5. Select **"Upload file"** — no wait, easier than that:

**Simpler method:**
1. Go to **https://github.com/alangner69/dcs**
2. Click **"Add file"** → **"Upload files"**
3. Drag your downloaded **index.html** into the box
4. Scroll down and click **"Commit changes"**
5. Your website updates within about 60 seconds ✅

## Changing the password

Open `index.html` in any text editor, search for:

```
dorecanine2025
```

And replace it with your new password. Then upload the file to GitHub as above.

---

## Adding or swapping photos

Photos need to be uploaded to GitHub separately.

1. Go to **https://github.com/alangner69/dcs**
2. Click **"Add file"** → **"Upload files"**
3. Upload your new photo (keep the filename simple, no spaces — e.g. `winnie-park.jpg`)
4. Commit the change

Then to show it on the site, you'll need to update the photo filename in `index.html`. If you're not sure how, just ask for help.

---

## Something gone wrong?

GitHub keeps a full history of every version of the site. If something breaks, you can always roll back. Just ask for help.
